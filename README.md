# City Care Web

Urban Company inspired service booking platform.

## Features
- User Authentication
- Vendor Dashboard
- Admin Panel
- Booking Flow
- Slot Selection
- Payment Integration
- Auto Cancel Cron
- Role Based Access

## Tech Stack
Frontend: React + Vite  
Backend: Node.js + Express  
Database: MySQL  

## Folder Structure
- /src → Frontend
- /backend → API Server
